# Code Beispiel: ML mit Scikitlearn
Dies ist ein Code Beispiel, das in Jupiter Notebook geschrieben wurde.
## Übersicht
Dieses Skript demonstriert maschinelles Lernen mit Scikit-learn in einem Jupyter Notebook.

## Voraussetzungen
- Python 3.x
- Jupyter Notebook
- Scikit-learn
- NumPy
- Pandas
- Matplotlib

## Installation
```bash
pip install jupyter scikit-learn numpy pandas matplotlib
```

## Verwendung
1. Jupyter Notebook starten: `jupyter notebook`
2. Die Notebook-Datei öffnen
3. Zellen nacheinander ausführen

## Funktionen
- Datenvorverarbeitung
- Modelltraining
- Evaluierung und Visualisierung
